# Curriculum & Branching Strategy

## Branching Strategy
- **Source of Truth**: `main`
- **Topic Branches**: Each topic or module will have its own branch following the naming convention `topic/{number}-{topic-name}`.
- **Workflow**: 
  1. Create a topic branch from `main`.
  2. Implement changes in the topic branch.
  3. Merge back to `main` upon completion (or as per bootcamp progression).
